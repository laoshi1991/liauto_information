# 理想汽车南向资金持股分析

本项目用于抓取并展示理想汽车（02015.HK）的南向资金持股数据。

## 功能

1.  **数据抓取**：每天定时（交易日 20:30）从东方财富/证券之星接口抓取最新的南向持股数据。
2.  **数据存储**：数据存储在 `理想南向持有.xlsx` 文件中。
3.  **数据展示**：提供 Web 页面展示持股趋势图（双坐标轴：净增持/总持股）。

## 运行

1.  安装依赖：
    ```bash
    pip install -r requirements.txt
    ```

2.  启动应用：
    ```bash
    python3 app.py
    ```

3.  访问：
    打开浏览器访问 `http://127.0.0.1:5001/`
